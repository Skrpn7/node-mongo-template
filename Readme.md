# 🚀 Node Mongo Backend Template

A production-ready scalable backend template built using **Node.js, Express, and MongoDB (Mongoose)**.

This template helps you start new backend projects quickly without repeating boilerplate setup.

---

## 📦 Features

✅ Clean folder architecture  
✅ MongoDB connection utility  
✅ Async error handler  
✅ API response & error wrapper  
✅ Environment variable setup  
✅ Health check route  
✅ Prettier configuration  
✅ Ready for authentication & feature expansion  
✅ Template-friendly structure  

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB Atlas (Mongoose)
- dotenv
- Prettier
- Nodemon
